# Postix

Ferramenta web para geração de posters com divisão automática em múltiplas páginas, utilizando medidas reais e exportação em PDF otimizado.

---

## 🚀 Funcionalidades

- Upload de imagens
- Divisão automática em múltiplas páginas
- Ajuste de medidas reais
- Exportação em PDF
- Interface intuitiva
- Organização para impressão de posters

---

## 💻 Tecnologias

- HTML
- CSS
- JavaScript
